Complete List of Project Fields

Core Project Information
- id - Project ID (auto-generated)
- name - Project name (derived from street + suburb)
- status - Project Status: ["Design Phase", "Construction Phase", "Cancelled", "Complete"]
- stream - Project stream
- suburb - Suburb
- street - Street address
- state - State (VIC, QLD, etc.)
- year - Project year (stored as date, year extracted)
- updated_at - Last update timestamp

Financial Fields
- deposit - Deposit amount
- project_cost - Project cost

- phone - Primary client phone
